# Соответствие требованиям ВКР

| Требование / идея ВКР | Реализация в MVP |
| --- | --- |
| Digital platform concept | Панель управления, сайдбар, единый workflow и локальное состояние |
| Russia/Zambia comparison | Переключатель компаний, юрисдикционные бейджи, валюты RUB и ZMW |
| Import from accounting systems | XLSX/CSV импорт и демо-данные, имитирующие выгрузку из учётной системы |
| Transformation to IFRS | Маппинг локальных счетов и движок учебных IFRS-корректировок |
| Analytical tools | Страница аналитики с коэффициентами и Recharts-графиками |
| Reporting | SOFP, P&L, Cash Flow на странице отчётов |
| ERP/XBRL roadmap | XML/XBRL-like экспорт, база знаний и roadmap в README |
| Security/access | Mock roles: аналитик, менеджер, аудитор; журнал действий |

## Научно-практический смысл

MVP показывает, как цифровая платформа может связать импорт локальных данных, методологический маппинг, трансформационные корректировки, отчётность и аналитический слой. Расчёты упрощены, но workflow соответствует логике внедрения IFRS/MSFO в образовательном сценарии.
