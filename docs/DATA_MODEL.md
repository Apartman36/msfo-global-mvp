# Модель данных

Типы определены в `lib/types.ts`.

## Company

Компания содержит идентификатор, название, юрисдикцию `RU` или `ZM`, страну, валюту, презентационную валюту, описание и отрасль.

## TrialBalanceEntry

Строка ОСВ содержит код и название счёта, тип счёта, дебет, кредит, сальдо, валюту и опциональные заметки. Для контрактивных счетов, например амортизации, дебет/кредит используются для корректного знака в отчётах.

## IfrsLine

IFRS-строка содержит код, тип отчёта `SOFP`, `PL` или `CF`, раздел, русское и английское название, порядок сортировки.

## AccountMapping

Связывает локальный счёт с IFRS-строкой. Тип маппинга бывает `auto` или `manual`. Ручное изменение фиксируется в журнале действий.

## AdjustmentEntry

Трансформационная проводка содержит стандарт, описание, дебетуемую и кредитуемую IFRS-строку, сумму, валюту, область влияния, объяснение и дату создания.

## FinancialStatementLine

Отчётная строка хранит локальную сумму, сумму корректировки и итоговую IFRS/MSFO сумму.

## AuditEvent

События фиксируют время, роль, действие, объект и детали. Это учебный append-only журнал в localStorage, не система безопасности.

## Persistence

Fixtures загружаются из TypeScript-файлов:

- `companies.ts`
- `trial-balances.ts`
- `mappings.ts`
- `ifrs-lines.ts`

После первого изменения состояние сохраняется в localStorage. Production database в v1 отсутствует.
